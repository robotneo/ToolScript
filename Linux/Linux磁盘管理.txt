磁盘管理
	df 查看磁盘分区使用状况
		参数:
		-l	仅显示本地磁盘(默认)
		-a	显示所有文件系统的磁盘使用情况，包括比如/proc/
		-h	以1024进制计算最适合的单位显示磁盘容量
		-H	以1000进制计算最适合的单位显示磁盘容量
		-T	显示磁盘分区类型
		-t	显示指定类型文件系统的磁盘分区
		-x	不显示指定类型文件系统的磁盘分区
	du 统计磁盘上文件的大小
		参数:
		-b	以byte为单位统计文件
		-k	以KB为单位统计文件
		-m	以MB为单位统计文件
		-h	按照1024进制以最适合的单位统计文件
		-H	按照1000进制以最适合的单位统计文件
		-s	指定统计目标
磁盘分区和格式化
	1.主分区和扩展分区总数不能超过四个
	2.扩展分区只能有一个
	3.扩展分区不能直接存储数据
	支持热插拔(服务器)
分区
	MBR分区(主分区不能超过四个，一个分区最多2TB)(服务器自动识别硬盘，但是需要分区挂载):
	fdisk分区工具，也就是fdisk命令(交互模式和命令模式)
	fdisk命令	硬盘分区工具
		参数:
		-l	列出当前服务器上磁盘的信息。
		fdisk 加磁盘位置	进入分区工具
			m	查看帮助信息
			p	查看分区信息
			n	添加新分区
				p	主分区
				(进入主分区，并给出一个分区编号，从哪个扇区开始(起始扇区)，
				并指定分区大小(终止扇区))，指定分区的大小
				e	扩展分区
			d	删除分区
			
			
	GPT分区(主分区几乎没有限制(128主分区)，每个分区大小几乎(18EB(EB-PB-TB-GB))没有限制):
	parted	GPT分区工具，也是命令(交互模式和命令模式)
	参数:进入parted
		help	帮助
		select /dev/sdc(目标硬盘)	指定磁盘
		mklabel msdos/gpt	(MBR(msdos),GPT(gpt))指定分区表的类型
		print all	查看所以磁盘分区类型(MBR/GPT)
		mkpart	添加新分区：
			询问卷标(分区名称) 询问文件系统(格式化)
			询问开始(从第几兆B开始)，结束(从第几兆B结束)，(4K对齐)cencal
			一般牺牲前面的一兆，第一兆B开始到2000兆B结束。
			实例: mkpart test 2000 3000
			数据重叠(就发出警告)。
		rm 分区编号		删除分区
		unit GB		指定分区单位分区
		quit	退出交互模式
		
分区格式化:
	mkfs	分区格式化，写入文件系统
		mkfs.ext4 /dev/sdb1		写入文件系统，格式化
		mkfs -t ext3 /dev/sdb2	和上面的格式化一样，写法不同

挂载:
	分区没有挂载是不能使用的
	mkdir /mnt/挂载目录		一般设在/mnt目录下的挂载点
	mount	挂载
	umount  卸载
	不具有永久性的，系统重启又需要挂载，所以需要写入配置文件，开机自启动
	/etc/fatab配置文件，在文件的末尾写入对于的数据
	规则:一般为设备名称(设备目录) 挂载点目录	文件类型	写入权限(defaults)	0	0
